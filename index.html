Mission statement

The aim of this project is to analyze League of Legends match data to derive relevant statistics. This is tool is designed to increase end users' win percentages in League Of Legends ranked games. This tool is still under construction.

Methodology

The backend is composed of Python modules that query Riot Games API (available here)
The data is entered into several MongoDB databases.
The frontend is composed of JavaEE web pages (JSPs and JavaBean technology).
